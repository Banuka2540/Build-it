# Build-it

A responsive construction-service website where users can explore contractors, materials, and service plans. The site delivers a smooth, modern user interface with interactive features to make project planning easier and more accessible.

- **Live Demo:** [https://build-it-ashen.vercel.app](https://build-it-ashen.vercel.app)

## Features

- Browse a curated list of contractors and building materials
- View various construction service plans
- Modern, responsive UI for seamless experience on all devices
- Interactive features for project planning
- Built with accessibility and usability in mind

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Banuka2540/Build-it.git
   cd Build-it
   ```

2. **Open `index.html` in your browser:**
   You can open the project directly using your browser, as there is no backend/server-side requirement.
